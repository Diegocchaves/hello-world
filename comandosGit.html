Lista de Comandos en GIT

  • Comando para moverse entre carpetas: cd <path>
  • Configurar Nombre que salen en los commits: git config --global user.name "Diegocchaves"
  • Configurar Email: git config --global user.email diegocchaves21@gmail.com
  • Marco de colores para los comando: git config --global color.ui true

Iniciando repositorio

  • Iniciamos GIT en la carpeta donde esta el proyecto: git init
